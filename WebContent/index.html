Estrai archivio

Estrai i contenuti del file .7zip in una parte accessibile del dispositivo

Apri il cmd(Windows)

Su Windows premi il tasto Windows+R e scrivi cmd o naviga nel menù Start, scrivi esegui e poi scrivi cmd

Naviga nel cmd

Con il comando 'cd' naviga fino alla cartella dove hai estratto il file i contenuti del .7zip se vuoi essere sicuro di essere nella cartella giusta, il comando 'dir' ti restituirà una lista con gli elementi presenti dentro la cartella

Esegui il programma

Con il comando 'java -jar bitcamp_superMarketManager.jar' eseguirai il programma con un nome del supermercato di default, se vuoi aggiungere il nome del tuo supermercato ti basta aggiungere il nome subito dopo 'bitcamp_superMarketManager.jar' ad esempio 'java -jar bitcamp_superMarketManager.jar Eurospin'

Usa il programma

Con l'interfaccia intuitiva potete finalmente utilizzarlo per i vostri scopi
